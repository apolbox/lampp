Selamat datang di Lampp webserver.

Sebuah paket instalasi webserver yang digunakan untuk mengembangkan aplikasi berbasis web. Paket yang tersedia saat ini:
  1. apache2 (Linux)
  2. lighttpd (Android)
  3. mariadb (Android)
  4. mysql (Linux)
  5. php (All)
  6. phpmyadmin (All/Optional)
Anda dapat menemukan repositorinya di https://github.com, Untuk mengunduhnya Anda dapat mengunjungi link dibawah ini:
